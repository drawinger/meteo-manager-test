# meteo-api

Сервер, отвечающий получение метеоданных с API


# Установка

* Создаем виртуальное окружение:
    ```
    python3 -m venv venv
    ```
* Активируем окружение:
    ```
    source venv/bin/activate
    ```
* Устанаваливаем зависимости:
    ```
    pip install -r requirements.txt
    ```
* Запускаем сервер:
    ```
    python main.py
    ```
